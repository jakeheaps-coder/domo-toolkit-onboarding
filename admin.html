Domo Toolkit

Build landing pages, generate images, create videos, and more — all with AI. No coding required.

Follow the steps below to get started
1

Request Access to Claude Code

Click below to request your Claude Code license through the Domo App Studio. This gives you access to the AI assistant that powers the toolkit.

Request Claude Code Access
You'll receive an email from Anthropic once your access is approved. This usually takes 1-2 business days.
2

Check Your Email & Create Your Account

Once approved, you'll get an email from Anthropic with instructions to create your Claude account.

  1. Open the email from Anthropic (check spam if you don't see it)
  2. Click the signup link in the email
  3. Create your account with your @domo.com email
  4. Choose a secure password
3

Download VS Code

Claude Code runs inside VS Code — Microsoft's free code editor. Don't worry, you won't need to write any code. This is just where Claude lives.

Download VS Code
You will need admin access to do this. Click the check mark in your top bar, request administrator access, reason: "VS Code Download"
4

Install the Claude Code Extension

Once VS Code is installed, add the Claude Code extension so you can use the AI assistant.

  1. Open VS Code
  2. Click the Extensions icon in the left sidebar (or press Cmd+Shift+X)
  3. Search for "Claude Code"
  4. Click "Install" on the official Anthropic extension
  5. Click the Claude icon in the sidebar to open it
  6. Sign in with the Anthropic account you created in Step 2
5

Create a GitHub Account

GitHub is where the toolkit lives. You need a free account to access it. Think of it as a shared drive for code.

Create GitHub Account
  1. Sign up with your @domo.com email
  2. Choose a username (this is what you'll share in the next step)
  3. Verify your email by clicking the link GitHub sends you
Already have a GitHub account? Skip this step and use your existing username.
6

Get Access to the Toolkit

Enter your GitHub username below and Jake will add you to the private toolkit repository.

github.com/
Request sent! Jake will review and grant access within 24 hours.
You'll get a GitHub notification email when access is granted. Then continue to Step 7.
7

Set Up Your Toolkit

Once Jake grants you access, open Claude Code in VS Code and paste this message. Claude will handle everything from there.

Clone the creative director toolkit from https://github.com/jakeheaps-coder/domo-toolkit and run the setup script

Claude will automatically:

  1. Download the toolkit to your computer
  2. Install all the tools you need
  3. Connect you to the Domo messaging database
  4. Set up AI image and video generation
  5. Configure daily auto-updates
The setup takes about 5 minutes and walks you through everything step by step. Just follow Claude's prompts.
